# Behind PRISMA's Success: GAI's Support in Action

This repository contains the code and research results connected to the scientific article titled "Behind PRISMA's Success: GAI's Support in Action".

## Overview

This project explores the role of Generative AI (GAI) in supporting the PRISMA (Preferred Reporting Items for Systematic Reviews and Meta-Analyses) methodology.

## Repository Structure

```
Behind-PRISMA-s-Success/
├── code/               # Source code and scripts
├── data/               # Research data and datasets
├── docs/               # Documentation and additional resources
├── results/            # Research results and analysis outputs
└── notebooks/          # Jupyter notebooks for analysis
```

## Getting Started

### Prerequisites

- Python 3.8 or higher
- pip (Python package manager)

### Installation

```bash
# Clone the repository
git clone https://github.com/mq3n/Behind-PRISMA-s-Success.git
cd Behind-PRISMA-s-Success

# Install dependencies
pip install -r requirements.txt
```
