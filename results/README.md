# Results Directory

This directory contains research results, analysis outputs, and generated artifacts.

## Structure

- `figures/` - Generated plots, charts, and visualizations
- `tables/` - Output tables and statistical summaries
- `reports/` - Generated reports and analysis documents
- `metrics/` - Performance metrics and evaluation results

## Guidelines

- Organize results by experiment or analysis date
- Include metadata about how results were generated
- Version control small result files (figures, summary tables)
- Use external storage for large output files
- Document any results referenced in publications
