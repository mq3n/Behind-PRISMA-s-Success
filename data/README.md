# Data Directory

This directory contains research data and datasets used in the project.

## Structure

- `raw/` - Original, unprocessed data
- `processed/` - Cleaned and processed data ready for analysis
- `examples/` - Small example datasets for testing and demonstration

## Important Notes

- **Large data files should not be committed to the repository**
- Use `.gitignore` to exclude large files
- Consider using external storage (e.g., cloud storage, data repositories) for large datasets
- Document data sources and preprocessing steps
- Include data dictionaries or schema descriptions where applicable

## Data Privacy

- Ensure all data complies with privacy regulations
- Remove or anonymize any personal or sensitive information
- Document any data usage restrictions or licenses
